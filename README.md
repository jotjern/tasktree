This project is intended to provide an ergonomic todo list, representing it as a tree of tasks
I've had a vision for this site for a long time, but never had the time to make it, but now the SWE-agents are finally good enough to make it
It stores the todos in a private github gist, providing convenicence and data ownership
The app is available at tasktrees.com
